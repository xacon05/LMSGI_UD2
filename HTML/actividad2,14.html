Glosario Alfabético

API
Interfaz de Programación de Aplicaciones, un conjunto de funciones y procedimientos que permiten la creación de aplicaciones que acceden a características o datos de un sistema operativo, aplicación u otro servicio.
CSS
Hojas de Estilo en Cascada, un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en HTML o XML.
HTML
Lenguaje de Marcado de Hipertexto, el estándar para la creación de páginas web y aplicaciones web.
JavaScript
Lenguaje de programación que permite implementar funciones complejas en páginas web.
JSON
Notación de Objetos de JavaScript, un formato de texto ligero para el intercambio de datos.
XML
Lenguaje de Marcado Extensible, un lenguaje de marcado que define un conjunto de reglas para la codificación de documentos en un formato que es legible tanto para humanos como para máquinas.
